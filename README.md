# EC7212 Computer Vision - Assignment 1

This repository contains solutions for the four image processing tasks in Assignment 1.

## Requirements

- Python 3.6+
- OpenCV (`pip install opencv-python`)
- NumPy (`pip install numpy`)

```bash
